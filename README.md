# E-Commerce-Web-App

Java を使ったECサイトアプリ

<h3>機能</h3>
MVC (Model View Controller) - 各レイヤーでデータの流れを操作 </br>
</br>
Controllerレイヤー: クライアントからのリクエストを処理</br>
Servicesレイヤー: ビジネスロジックを実行</br>
Repositoryレイヤー: データベースへの接続</br>
</br>
CRUD処理で商品の追加、読み込み、更新、検索、削除が可能</br>

<h3>フレームワーク</h3>
- Spring Boot

<h3>フロントエンド</h3>
- React + Vite

<h3>データベース</h3>
- PostgreSQL
